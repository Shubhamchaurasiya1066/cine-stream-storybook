# Cine Stream - Storybook

A Storybook implementation for the Cine Stream project using Next.js and React.

## Tech Stack

- Next.js
- React
- Storybook
- Redux Toolkit
- Tailwind CSS

## Components

- Hero
- SearchBar
- MovieCard

## Features

- Reusable UI Components
- Args & Controls
- Redux Provider
- Light/Dark Theme

## Run

```bash
npm install
npm run storybook
```

## Build

```bash
npm run build-storybook
```
